Breakout Game on the Micro:bit

2019-09-24

A ‘breakout’ clone for the microbit. Probably the smallest (graphically speaking) version of the Breakout game, uses 25 pixels/leds (arranged as a 5x5 grid).

This is the repo/python code. Check the releases for a hex file you can copy to your own micro:bit.
